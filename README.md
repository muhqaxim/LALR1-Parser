# LALR(1) Parser

This project implements an LALR(1) parser in Python.

## Structure

- `src/`: Source code for the parser.
- `tests/`: Unit tests.
- `requirements.txt`: Project dependencies.

## Usage

1. Clone the repository.
2. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3. Run the parser:
    ```bash
    python src/main.py
    ```

## Testing

Run the tests:

pytest
